# Ritik-Tyagi-Tic-Tac-Toe
Tic-Tac-Toe-Game-web
Technologies: HTML, CSS (Flexbox), JavaScript (DOM Manipulation) 
Project Goal: Engineered a complete two-player, web-based Tic-Tac-Toe game with responsive design. 
Details: 
● Implemented game logic using JavaScript DOM manipulation. 
● Designed a clean and responsive layout using Flexbox. 
● Added win-condition detection, draw logic, and reset functionality. 
● Optimized UI for smooth user experience on both desktop and mobile.
